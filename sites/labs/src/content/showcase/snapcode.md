## What is SnapCode?

SnapCode is a full-featured Java IDE that runs in the browser. It is free, modern, powerful, and tailored for education — making it the fastest and easiest way to start writing and sharing Java code.

## Features

SnapCode offers modern coding features, support for Java REPL (Read-Eval-Print-Loop), support for working with projects in the cloud, and full-featured library support for UI, graphics, charting, 3D, and more.
