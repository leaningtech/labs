BrowserPod offers two subscription plans: **Personal** (free) and **Pro** (paid).

## Personal Plan (Free)

The Personal plan allows you to use BrowserPod for:

- Non-commercial purposes
- Technical evaluation
- Personal projects

**Requirements:**

- Your application must display BrowserPod attribution (visible link + logo)
- No resale or redistribution

## Pro Plan (Paid)

The Pro plan allows you to use BrowserPod for:

- Commercial purposes
- No attribution required

## Usage Limits

Both plans are subject to token consumption limits based on "boots" (instance starts). Usage is tracked via a global metering system tied to your API key. See the [Pricing Policy](https://browserpod.io/browserpod-pricing-policy) for specific limits.

## Examples

- **Example 1:** You're evaluating BrowserPod for a future project. → **Personal plan (free)**
- **Example 2:** You're building a personal/hobby project. → **Personal plan (free)**
- **Example 3:** You're a company building a customer-facing product. → **Pro plan required**
- **Example 4:** You're a company building an internal tool. → **Pro plan required**
